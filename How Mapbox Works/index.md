<p>Welcome to Mapbox! Mapbox is a developer platform used <a href="https://www.mapbox.com/industries" rel="nofollow">across industries</a> to create custom applications that solve problems with maps, data, and spatial analysis. Mapbox's tools are building blocks that support every part of the web and mobile map-making process. Whether your goal is to build a beautiful map to match your website or to build a full-featured geoprocessing application, we have you covered.</p>
<p>These guides will introduce you to the building blocks of Mapbox and how you can:</p>
<ul>
<li>Work with in Mapbox's <strong>robust data</strong>.</li>
<li><strong>Style</strong> your map down to the smallest details.</li>
<li><strong>Upload</strong> or <strong>create</strong> custom data.</li>
<li><strong>Develop</strong> full-featured web and mobile applications.</li>
<li><strong>Extend</strong> your app's functionality with web services for <a href="/mapbox/help/blob/publisher-production/help/glossary/geocoding">geocoding</a>, directions, spatial analysis, and more.</li>
<li><strong>Create</strong> static maps programmatically.</li>
</ul>
<h2><a id="user-content-use-mapbox-map-data" class="anchor" aria-hidden="true" href="#use-mapbox-map-data"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z"></path></svg></a>Use Mapbox map data</h2>
<p>{{

}}</p>
<p>Our core <a href="/mapbox/help/blob/publisher-production/help/glossary/tileset">tilesets</a>: Mapbox Streets, Mapbox Terrain, Mapbox Traffic, and Mapbox Satellite. Each tileset contains a unique set of data from a variety of sources.</p>
<ul>
<li><strong>Mapbox Streets</strong> includes streets, buildings, administrative areas, water, and land data based on <a href="http://www.openstreetmap.org/" rel="nofollow">OpenStreetMap</a>, updated as often as every five minutes.</li>
<li><strong>Mapbox Terrain</strong> includes landcover data and a worldwide elevation data set complete with contours, hillshade, and elevation data.</li>
<li><strong>Mapbox Satellite</strong>  includes global satellite imagery from <a href="https://www.mapbox.com/about/maps" rel="nofollow">a range of sources</a>, processed and seamed together by Mapbox.</li>
<li><strong>Mapbox Traffic</strong> includes regularly updated vehicle congestion information on top of Mapbox Streets.</li>
</ul>
<p>You can find a full list of layers available in the Mapbox Streets, Terrain, and Traffic sources in our <a href="https://www.mapbox.com/vector-tiles/" rel="nofollow">Vector Tiles overview</a>. You can read more about our data sources on our <a href="https://www.mapbox.com/about/maps" rel="nofollow">Maps page</a>.</p>
<p><a href="/mapbox/help/blob/publisher-production/help/how-mapbox-works/mapbox-data">{{}}</a></p>
<h2><a id="user-content-design-a-map" class="anchor" aria-hidden="true" href="#design-a-map"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z"></path></svg></a>Design a map</h2>
<p>Custom map design is one of the core functions of Mapbox Studio. We provide an advanced application for putting this customization at your fingertips with the Mapbox Studio style editor. In Mapbox Studio, you can start with one of our <a href="https://www.mapbox.com/maps/" rel="nofollow">template styles</a> or <a href="https://www.mapbox.com/designer-maps/" rel="nofollow">designer styles</a> and style each individual layer to your exact specification.</p>
<p>{{

}}</p>
<p>On the left you can see the <strong>Basic style</strong>, a template style made up of a limited set of layers. On the right you can see a <strong>custom style</strong> that uses all the same underlying data, but that has been customized by changing colors and fonts in Mapbox Studio style editor.</p>
<p>The <a href="https://www.mapbox.com/studio/styles" rel="nofollow">Mapbox Studio style editor</a> is a full-featured map editor that gives you total control over the style of your map directly in your browser. Whether you start with a Mapbox template style or start from scratch, the styling possibilities are virtually endless. With the Mapbox Studio style editor, you can:</p>
<ul>
<li>Create styles that change dynamically based on zoom level.</li>
<li>Use custom fonts.</li>
<li>Set custom alignment, pitch, offset, and more for your labels.</li>
<li>Set colors, weights, and opacity for your map layers.</li>
<li>Filter tilesets based on attributes.</li>
</ul>

<p>Mapbox provides robust geospatial data with our Streets, Terrain, Traffic, and Satellite tilesets, but mapping applications often require custom data. To add custom data to your map, you can upload your own data as <a href="/mapbox/help/blob/publisher-production/help/glossary/tileset">tilesets</a> or create <a href="/mapbox/help/blob/publisher-production/help/glossary/dataset">datasets</a>.</p>
<p><strong>Tilesets and datasets are two different types of data</strong>: tilesets are styleable and datasets are editable. Styling includes changing things like color, opacity, font, or icon. Editing includes changing the placement of features (points, lines, polygons), their geometries, and adding or deleting features from a feature collection. If you have created or imported a dataset, you can export your dataset to a tileset right in Mapbox Studio and use it in the Mapbox Studio style editor like you would any other tileset.</p>
<h3><a id="user-content-upload-tilesets" class="anchor" aria-hidden="true" href="#upload-tilesets"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z"></path></svg></a>Upload tilesets</h3>
<p>Tilesets are lightweight collections of vector data that are optimized for rendering and are not editable. When you <a href="/mapbox/help/blob/publisher-production/help/troubleshooting/uploads">upload custom data</a>, your files are converted to vector tilesets, which can be styled in the Mapbox Studio style editor, added to <a href="https://www.mapbox.com/mapbox-gl-js/example/queryrenderedfeatures/" rel="nofollow">interactive web maps with Mapbox GL JS</a>, and used in mobile applications created with the Mapbox Maps SDKs for <a href="https://docs.mapbox.com/ios/maps/overview/" rel="nofollow">iOS</a> and <a href="https://docs.mapbox.com/android/maps/overview/" rel="nofollow">Android</a>.</p>
<p>For more details about accepted file types and methods for uploading data, see the <a href="/mapbox/help/blob/publisher-production/help/how-mapbox-works/uploading-data">Uploading data guide</a>.</p>

<p>A dataset is an editable collection of <a href="https://tools.ietf.org/html/rfc7946" rel="nofollow">GeoJSON features</a>. A feature stored with Mapbox has both geometries and properties (attributes), both of which can be edited in the Mapbox Studio dataset editor or through the Mapbox Datasets API. You can use the <a href="https://www.mapbox.com/studio-manual/reference/datasets" rel="nofollow">dataset editor</a> in Mapbox Studio to import, create, and edit GeoJSON point, line, and polygon features and their properties. Once you've finished working with your dataset, you can export it to a tileset for use with the Mapbox Studio <a href="https://www.mapbox.com/studio/" rel="nofollow">style editor</a>. There are limits to how much data you can load into the Mapbox Studio dataset editor at a time, but you can use the Mapbox <a href="https://docs.mapbox.com/api/maps/#datasets" rel="nofollow">Datasets API</a> to add more features and manage them programmatically.</p>

<p><a href="https://www.mapbox.com/mapbox-gl-js/" rel="nofollow">Mapbox GL JS</a> is a JavaScript library for creating interactive, customizable maps from Mapbox styles and vector tiles.
